# 画面遷移図（詳細）

## フェーズ1〜4（個人利用）

ログイン画面を起点とし、認証後にボード画面を中心としたタスク管理画面群へ遷移する。

```mermaid
flowchart LR
    S04[S-04<br/>ログイン画面]
    S03[S-03<br/>ユーザー登録画面]
    S01[S-01<br/>ボード画面]
    S02[S-02<br/>カード詳細画面]
    S05[S-05<br/>完了タスク一覧画面]

    S04 -- 新規登録リンク --> S03
    S03 -- 登録完了 / 戻る --> S04
    S04 -- ログイン成功 --> S01
    S01 -- ログアウト --> S04
    S01 -- カード選択（モーダル表示） --> S02
    S02 -- 保存 / キャンセル（モーダル閉じる） --> S01
    S01 -- 完了タスク一覧へ --> S05
    S05 -- 戻る --> S01
    S05 -- 行クリック（モーダル表示） --> S02
    S02 -. モーダル閉じる .-> S05
```

**補足：**
- フェーズ1〜2は認証機能がないため、起点はボード画面（S-01）となり、S-03/S-04 は存在しない
- フェーズ3でログイン・ユーザー登録・ログアウトの遷移が追加される
- フェーズ4で履歴検索画面（S-05）への遷移が追加される
